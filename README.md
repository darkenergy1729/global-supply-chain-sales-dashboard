# global-supply-chain-sales-dashboard
Global Supply Chain & Sales Dashboard
Project Overview

This project analyzes sales performance, profitability, product performance, and regional trends using Microsoft Excel.

The dashboard provides interactive business insights through Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Dashboarding
KPI Analysis
Data Visualization
Business Objectives
Analyze regional revenue performance
Identify top-performing products
Evaluate category profitability
Track monthly sales trends
Support business decision-making through data insights
Dashboard KPIs
Total Revenue: 30.5M
Total Profit: $75K
Average Profit Margin: 1.06%
Total Quantity Sold: 400
Dashboard Features
Revenue by Region
Profit by Category
Monthly Revenue Trend
Top 10 Products by Revenue
Interactive Slicers
Business Recommendations
Key Insights
North America generated the highest revenue contribution.
CPU products achieved the highest profit contribution.
February recorded peak monthly revenue.
G.SKILL Ripjaws V Series generated the highest revenue.
South America contributed the lowest revenue.
Business Recommendations
Increase inventory allocation for high-performing CPU products.
Focus marketing efforts in North America.
Improve sales strategies in low-performing regions.
Monitor seasonal demand spikes.
