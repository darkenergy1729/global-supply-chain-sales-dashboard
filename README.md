# 📊 Global Supply Chain & Sales Dashboard

## Overview

Interactive Excel dashboard analyzing sales performance, profitability, product categories, and regional revenue trends.

### Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* KPI Analysis

### Key Insights

* North America generated the highest revenue.
* CPU products delivered the highest profit.
* February recorded peak monthly revenue.
* G.SKILL Ripjaws V Series was the top revenue-generating product.

### Business Recommendations

* Increase inventory allocation for high-performing CPU products.
* Focus marketing efforts in North America.
* Improve sales strategies in low-performing regions.
